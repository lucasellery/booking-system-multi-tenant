# Booking System

## Running the project

First, ensure you have `pnpm` installed and run the commando bellow on the terminal:

```
pnpm dev
```
_if you do not have `pnpm` installed, follow the instalation guide: https://pnpm.io/pt/installation_


### After run de project:

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
